# ATM-Java 🏧

## Un simulator de **ATM** realizat in Java.
Contine:
- Login & Register
- Operatii bancare
  > Retrageri & depuneri
- Posibilitatea de a face un imprumut
- Bonusuri financiare obtinute in urma depunerilor substantiale
- Verificari in vederea retragerilor
- Verificari in vederea imprumuturi
- Praguri limita
